About Menu Actions

File

Many of the menu items are standard on most programs. The ones that I have added are:

  • Open File -- Open a previously saved xml file
  • Save File -- Save the file after edits have been made.
  • Save File As... -- Save a copy of the data to a new file, leaving the old file unchanged.
  • Revert to Saved -- Discard any edits you've made and re-load the previously saved file.
  • Save Outline -- Saves the coordinates of the outline (i.e. the surface being cut) to a tab-delimited text file. Point spacing is based on the curve resolution. This can be read into most spread-sheet programs. There are columns for X and Z coordinates as well as the diameter and circumference at each point. This allows the user to utilize a spreadsheet for other calculations relative to the surface of the object.
  • Page Setup... -- Set page orientation for printing
  • Print... -- Prints the currently active window. It's best to first click in the window you want to print to make sure that window is currently the active window.

Outline

  • 2Pts: Vertical -- If you only have 2 points, this will make them perfectly vertical.
  • 2Pts: Horizontal -- If you only have 2 points, this will make them perfectly horizontal.
  • Offset Top to Zero -- Sets the top part of the cutter's path to zero.
  • Offset Bottom to Zero -- Sets the bottom part of the cutter's path to zero.
  • Invert -- Inverts the shape (i.e. turns it upside-down).
  • Clear All -- Clears the outline (for a fresh start).
  • Zoom to Fit -- Fit the outline in the window.

View3D

  • Set Bowl Color... -- Change the color of the bowl. This setting is saved when you exit the program and restored the next time.
  • Set Background Color... -- Change the color of the background. This setting is saved when you exit the program and restored the next time.
  • Show Grid --Enable/disable the grid. This setting is saved when you exit the program and restored the next time.
  • Save 3D View... -- Save the current view in the window as a jpeg file.

Other Menu Items

Most of the other Menu items are similar to other programs you have used. If in doubt, try it out and see what happens.
