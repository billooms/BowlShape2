About View3D

The 3D View window is for viewing the 3D bowl..

Rotate View or Bowl

When rotating the View, the bowl stays in a fixed position on the grid. Using the mouse, you can move around the view (i.e. like walking around the bowl that is setting on a table). This mode works best seeing the outside of a shape.

When rotating the Bowl, the grid remains fixed. Using the mouse, you can move the bowl (i.e. like holding the bowl in your hand). This mode works best for seeing the inside of a shallow bowl. Note that for a flat surface, you need to move the piece to get a good reflection from one of the light sources.

Using the mouse

Click with the mouse and move it around while holding the button down. This lets you rotate the view around. You can move the view left/right/up/down by using the right button (Mac use Apple-click). You can zoom in closer (or further away) by using your mouse scroll-wheel (Mac use option-click and move the mouse up or down).

While rotating the view, you’ll notice that the grid moves as well, and the light source remains above the bowl. Another option is to rotate the bowl with respect to the grid and light source -- click the pop-up menu in the lower right corner of the screen and select “Bowl” and try moving the bowl around. This is good to see the underside features ofthe bowl. In this mode, right-click moves the bowl left/right/up/down (Mac apple-click), and alt-click (Mac option-click) moves the bowl closer or further away.

The “Restore View” button always brings you back to the starting default viewpoint.

View3D Menu actions

"Set Bowl Color..." is used to change the color of the bowl. This setting is saved when you exit the program and restored the next time.

"Set Background Color..." is used to change the color of the background. This setting is saved when you exit the program and restored the next time.

"Show Grid" allows you to enable/disable the grid. This setting is saved when you exit the program and restored the next time.

"Save 3D View..." allows you to save the current view in the window as a jpeg file.
